# 개인정보처리방침 (JLR일산서비스 유틸)

최종 수정일: 2026-06-23

JLR일산서비스 유틸(이하 "본 확장 프로그램")은 One DMS(`https://www.jlrkdms.co.kr/`) 정비 관리 시스템에서 사용하는 내부 업무용 Chrome 확장 프로그램입니다.

## 1. 수집하는 정보

본 확장 프로그램은 사용자의 개인정보를 수집, 저장, 전송하지 않습니다.

본 확장 프로그램이 저장하는 정보는 다음과 같습니다.

- 내부의견 48시간 점검 문구(1~12번) 설정값
- 작업그룹 입력 항목 설정값

위 정보는 사용자가 직접 입력한 업무용 텍스트로, `chrome.storage.local`을 통해 **사용자의 로컬 브라우저 내부에만** 저장됩니다. 외부 서버로 전송되지 않으며, 개발자를 포함한 제3자가 접근할 수 없습니다.

## 2. 권한 사용 목적

- **storage**: 위 설정값을 저장하고 불러오기 위해 사용합니다.
- **호스트 권한 (`https://www.jlrkdms.co.kr/*`)**: One DMS RO 상세 화면의 내부의견·작업그룹 표에 반복 입력 항목을 자동으로 채워주는 기능을 구현하기 위해 필요하며, 해당 도메인 외 다른 사이트에는 접근하지 않습니다.

## 3. 제3자 제공

본 확장 프로그램은 어떠한 데이터도 제3자에게 제공, 판매, 공유하지 않습니다.
