# JLR Ilsan Helper 설치 설명서

## 1. 설치 전 준비

Chrome 브라우저를 사용해야 합니다.

제한판을 설치할 경우 아래 폴더를 사용합니다.

`사용제한_2026-06-30`

이 제한판은 **2026년 6월 30일 23:59:59(KST)** 까지만 사용할 수 있습니다.

## 2. 확장프로그램 설치 방법

1. Chrome을 실행합니다.
2. 주소창에 아래 주소를 입력합니다.

   `chrome://extensions`

3. 오른쪽 위의 **개발자 모드**를 켭니다.
4. 왼쪽 위의 **압축해제된 확장 프로그램을 로드합니다** 버튼을 누릅니다.
5. 설치할 확장프로그램 폴더를 선택합니다.

   제한판 설치 시 선택할 폴더:

   `내부의견_확장프로그램\사용제한_2026-06-30`

6. 확장프로그램 목록에 **JLR Ilsan Helper** 가 표시되면 설치가 완료된 것입니다.

## 3. One DMS에서 적용하는 방법

1. One DMS 사이트에 접속합니다.

   `https://www.jlrkdms.co.kr/`

2. 이미 열려 있던 One DMS 화면이 있다면 새로고침합니다.
3. RO 상세 화면의 **내부의견** 영역으로 이동합니다.
4. 기존 버튼 근처에 아래 버튼들이 보이면 정상 적용된 것입니다.

   - `48시간 일괄추가`
   - `일괄삭제`
   - `설정`

## 4. 1~12번 문구 설정 방법

1. Chrome 상단의 확장프로그램 아이콘을 누릅니다.
2. 또는 One DMS 내부의견 영역에 있는 **설정** 버튼을 누릅니다.
3. 팝업 설정창에서 1~12번 문구를 수정합니다.
4. **설정 저장** 버튼을 누릅니다.
5. 이후 `48시간 일괄추가`를 누르면 저장한 문구가 적용됩니다.

## 5. 사용 방법

### 일괄추가

RO 상세 화면의 내부의견 영역에서 **48시간 일괄추가** 버튼을 누릅니다.

누락된 항목만 추가되며, 이미 같은 항목이 있으면 중복으로 추가하지 않습니다.

### 일괄삭제

**일괄삭제** 버튼을 누르면 이 확장프로그램이 추가한 1~12번 내부의견만 삭제합니다.

다른 내부의견은 삭제하지 않습니다.

## 6. 업데이트 방법

새 버전 폴더를 받은 경우:

1. `chrome://extensions`로 이동합니다.
2. 기존 확장프로그램 카드에서 **제거**를 누릅니다.
3. 새 폴더 기준으로 다시 **압축해제된 확장 프로그램을 로드합니다**를 진행합니다.
4. One DMS 화면을 새로고침합니다.

같은 폴더 안의 파일만 수정된 경우:

1. `chrome://extensions`로 이동합니다.
2. 해당 확장프로그램 카드의 새로고침 아이콘을 누릅니다.
3. One DMS 화면을 새로고침합니다.

## 7. 삭제 방법

1. `chrome://extensions`로 이동합니다.
2. **JLR Ilsan Helper** 확장프로그램 카드에서 **제거**를 누릅니다.
3. Chrome 확인창에서 제거를 한 번 더 누릅니다.

## 8. 문제 해결

### 버튼이 보이지 않는 경우

- One DMS 화면을 새로고침합니다.
- `chrome://extensions`에서 확장프로그램이 켜져 있는지 확인합니다.
- 설치한 폴더가 올바른지 확인합니다.
- One DMS 주소가 `https://www.jlrkdms.co.kr/`로 시작하는지 확인합니다.

### 설정 저장이 안 되는 경우

- 1~12번 문구 중 빈 항목이 있는지 확인합니다.
- 제한판의 사용기간이 만료되지 않았는지 확인합니다.

### 제한판이 만료된 경우

2026년 6월 30일 23:59:59(KST) 이후에는 제한판의 일괄추가, 일괄삭제, 설정 저장 기능이 비활성화됩니다.
