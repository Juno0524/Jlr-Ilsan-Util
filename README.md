# One DMS 48시간 내부의견 확장 프로그램

`https://www.jlrkdms.co.kr/` 안에서만 작동하는 Chrome 확장 프로그램입니다.

## 설치

1. Chrome 주소창에 `chrome://extensions`를 입력합니다.
2. 오른쪽 위의 **개발자 모드**를 켭니다.
3. **압축해제된 확장 프로그램을 로드합니다**를 누릅니다.
4. 이 폴더(`내부의견_확장프로그램`)를 선택합니다.
5. 열려 있던 One DMS 화면을 새로고침합니다.

## 사용

RO 상세 화면의 **내부의견** 영역에서 **48시간 일괄추가** 버튼을 누릅니다.
누락된 항목만 추가하므로 같은 문구가 이미 있으면 중복 등록하지 않습니다.

**일괄삭제** 버튼은 이 확장 프로그램의 1번부터 12번까지에 해당하는
내부의견만 제거하며, 다른 내부의견은 삭제하지 않습니다.

12건이 표에 추가된 것을 확인한 뒤, 기존 One DMS의 **저장** 버튼으로 최종 저장합니다.

## 문구 설정

Chrome 도구 모음의 확장 프로그램 아이콘을 누르면 기능 선택 화면(**내부의견**, **작업그룹**)이 열립니다.

**내부의견**을 선택하면 1~12번 문구를 수정할 수 있습니다.
수정 후 **저장**을 누르면 다음 일괄추가부터 적용되며, **기본값 복원**도 지원합니다.
6번 뒤의 원케어 가입 여부는 기존처럼 차량 정보에서 확인하여 자동 입력됩니다.

**작업그룹**은 현재 준비 중인 기능입니다.

## 작업그룹 빠른 추가

RO 상세 화면의 **작업그룹** 영역에서 **진단** / **소음** 버튼을 누르면
작업그룹 행이 추가되고 작업내용 칸에 "진단" 또는 "소음"이 자동으로 입력됩니다.
