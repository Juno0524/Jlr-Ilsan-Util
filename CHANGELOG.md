# 업데이트 이력

## v3.1

- 리셉션 리마인드 기능 추가
  - 리셉션 로그인 후 DMS 접수 화면에 `리마인드 시작/종료` 버튼을 표시하고, 예약 행별로 입고예정/입고완료/예약변경/취소/부재 상태를 선택할 수 있게 했습니다.
  - 리마인드 종료 시 선택한 예약자 상태를 날짜/시간대별 요약 문구로 정리하고 복사할 수 있는 팝업을 표시합니다.
- 관리자 기능 확장
  - 관리자 메뉴에서 설정관리 사이트를 바로 열 수 있게 했습니다.
  - 관리자 전용 토글로 `클레임` 버튼과 리마인드 버튼 표시 여부를 제어할 수 있게 했습니다.
- 원격 설정 동기화 개선
  - 설정관리 서버의 내부의견 문구를 설치/시작 시와 주기적으로 동기화하도록 변경했습니다.
- 다운로드 사유 빠른 입력 추가
  - 다운로드 사유 입력 팝업에 `예약지` 버튼을 추가해 동의 체크와 사유 문구 입력을 한 번에 처리할 수 있게 했습니다.

## v1.3.13

- 전체보기 작성 중 감지 기준 개선
  - 전체보기에서 아무것도 입력하지 않았는데도 작성 중으로 인식되는 문제를 수정했습니다.

## v1.3.12

- 진단 내용 입력 팝업 가독성 개선
  - 라벨, 입력값, 선택 버튼, 안내 문구의 색상과 굵기를 조정해 더 선명하게 보이도록 했습니다.

## v1.3.11

- 원케어 자동 감지 안내 표시 위치 조정
  - `(자동으로 감지하여 입력됨)` 문구는 실제 내부의견 행에는 저장하지 않고, 전체보기에서만 흐릿한 보조 안내로 표시합니다.

## v1.3.10

- 원케어 자동 감지 문구 표시
  - 6번 원케어 가입 여부 자동 입력값 뒤에 `(자동으로 감지하여 입력됨)` 문구를 추가했습니다.

## v1.3.9

- 전체보기 작성 중 닫기 확인 추가
  - 전체보기에서 수정 적용 전 변경사항이 있을 때 Esc, 닫기, 배경 클릭으로 나가려 하면 확인 팝업을 표시합니다.

## v1.3.8

- 내부의견 전체보기 편집 범위 제한
  - 전체보기 수정 시 각 항목의 번호/제목 앞부분은 고정하고 마지막 `-` 뒤 내용만 수정할 수 있게 했습니다.

## v1.3.7

- 내부의견 전체보기 현재 행 수정 적용
  - 전체보기에서 수정한 내용이 기본 설정에 저장되지 않고, 현재 DMS 화면의 해당 내부의견 행에만 적용되도록 했습니다.
  - 수정 후에는 기존 DMS 저장 버튼을 눌러 최종 저장하는 흐름을 유지합니다.

## v1.3.6

- 내부의견 전체보기 세로 목록 적용
  - 전체보기 팝업을 1열 세로 목록으로 변경해 1~12번을 위에서 아래로 확인할 수 있게 했습니다.

## v1.3.5

- 내부의견 전체보기 읽기 전용 복원
  - 전체보기 팝업의 수정/저장 기능을 제거하고 다시 확인 전용으로 변경했습니다.

## v1.3.4

- 내부의견 전체보기 편집 개선
  - `전체보기` 버튼을 가장 왼쪽으로 이동했습니다.
  - 전체보기 팝업에서 1~12번 문구를 수정하고 저장할 수 있게 했습니다.
  - 팝업 글자 크기와 행 정렬을 개선했습니다.

## v1.3.3
